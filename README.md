# Sqwizzey Flow

Нативное iOS-приложение для умного контроля личных финансов. Акцент на кредитные карты — отслеживание грейс-периодов, расчёт дедлайнов и оптимальное распределение платежей.

## О проекте

Sqwizzey Flow помогает не переплачивать банкам. Приложение отслеживает льготные периоды по каждой карте, понимает сколько и когда нужно внести, и **умно подсказывает** с какой дебетовой карты на какую кредитку перевести деньги — с учётом обязательных расходов на жизнь.

Все данные хранятся **локально на устройстве** (SwiftData). Никаких подключений к банкам, серверам или Apple Wallet.

## Возможности

**Реализовано**

- Учёт карт — дебетовые и кредитные с лимитом, долгом, грейс-периодом, днём выписки и платежа
- Транзакции с двойной записью (парные операции)
- Категории — 13 предустановленных + создание своих
- Грейс-периоды — автоматическая генерация выписок с дедлайнами и срочностью
- Умный калькулятор — распределяет деньги по приоритету (горящие выписки первыми)
- Регулярные расходы — резерв на жизнь при расчёте безопасной суммы к погашению
- Мягкое удаление с восстановлением

**В планах**

- Дашборд со сводкой по всем картам
- Умные обоснования рекомендаций
- Импорт выписок с автокатегоризацией
- Push-уведомления о дедлайнах
- Виджет, Face ID, локализация

## Стек

Swift 6 / SwiftUI / SwiftData / iOS 18+ / MVVM
