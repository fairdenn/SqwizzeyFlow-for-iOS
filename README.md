# Sqwizzey Flow

iOS-приложение для умного управления кредитными картами. Отслеживает грейс-периоды, рассчитывает дедлайны и подсказывает, как распределить деньги чтобы не платить проценты.

## Что делает

- Учёт кредитных и дебетовых карт с лимитами, долгом и грейс-периодом
- Автоматическая генерация банковских выписок с дедлайнами
- Умный калькулятор погашения — распределяет деньги по приоритету
- Учёт обязательных расходов при расчёте безопасной суммы к погашению
- Транзакции с двойной записью, мягкое удаление
- Все данные хранятся локально (SwiftData), без подключения к банкам

## Стек

- Swift 6 / SwiftUI
- SwiftData (локальное хранение)
- iOS 18+

## Запуск

Открыть `SqwizzeyFlow.xcodeproj` в Xcode 16+, выбрать симулятор или устройство, запустить.

## Статус

MVP в разработке.
